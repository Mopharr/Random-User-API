How to Install

CS6 & CC 2014-

Open the .zxp file with Adobe Extension Manager CC. Once the extension is installed, restart Photoshop and navigate to Window > Extensions > Random User Generator.

CC 2015+

Download the .zip file and follow the instructions in the README. Once the extension is installed, restart Photoshop and navigate to Window > Extensions > Random User Generator.

How to use

1. Use the toggle to generate male, female, or both gendered users.

2. Click and drag the profile photo into your Photoshop document. You can also right-click on the picture and select "Copy Image".

3. Highlight the name text and paste it into a text object on your document.

4. Press the "New Random User" button to generate a new random user
