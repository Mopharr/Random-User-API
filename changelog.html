Change Log

API Changes & Updates

Follow us @randomapi

Version 1.3

9/27/2019
  • Use native JS dates instead of moment for ~60% speed increase in date calculation
  • Email addresses are now transliterated
  • More street/city names for NL dataset
  • Add country name to location block
  • Capitalized proper nouns
  • Split street into number and name properties
  • Fixed Norwegian id generation bug
  • Fixed Switzerland phone number format
  • Fixed Danish CPR number calculation
  • Restructured Random User Generator with ES6 classes
  • Added proper unit testing and CI

Version 1.2

6/17/2018
  • Fixed DOB and Registration dates to use ISO 8601 standard (yyyy-mm-ddThh:mm:ssZ)
  • DOB and Registration dates are now objects that include age
  • Street numbers range from 1-9999
  • Add coordinates to location
  • Add timezone to location
  • Add uuid to login
  • Fix AVS number format for CH
  • Add Norway data
  • Fix CA postal code format

Version 1.1

8/18/2016
  • Fixed HETU for FI
  • Fixed salt not being used in calculation for hashes (also hotfixed 1.0)
  • DOB and Registered use ISO 8601 time standards
  • DOB and Registered can't overlap in impossible ways
  • Added customizable password options

Version 1.0

4/7/2016
  • The Randomuser.me website and API are now running on Node.js!
  • Added inc & exc parameters to include/exclude specific fields
  • Multiple nats can be requested at a time in a comma separated list
  • Field names are now consistent across nats
  • Added prettyjson format option for human readable JSON
  • Increased results cap to 5,000 users per request
  • Added pagination support with page parameter
  • Added DK and TR

Version 0.8

2/4/2016
  • Random User Generator is open source!
  • Added BR, CA, CH, DE, IR, and NZ

Version 0.7

7/27/2015
  • Huge performance update
  • You can now generate up to 2,000 users per request now

Version 0.6

5/8/2015
  • Fixed seed bug that didn't respect nationalities
  • Seed is returned on 1st result only and is meant to regenerate a request with multiple results
  • Added AU, ES, FI, FR, IE, and NL
  • Fixed GB phone numbers, zipcodes, and changed SSN to NINO
  • RandomAPI members can now generate up to 1,000 results per call, Learn more

Version 0.5

3/3/2015
  • Added nationalities

Version 0.4.1

9/12/2014
  • Added new picture options (medium and thumbnail)
  • New formats added (sql, csv, and yaml)

Version 0.4

5/3/2014
  • Redesigned and rebranded the Random User API website and platform
  • We doubled our profile photo database! We now return 100 male and 100 female portraits
  • Salts now use strictly alphanumeric characters
  • Restructured our file system to avoid redundancy
  • Optimized generator

Version 0.3

1/14/2014
  • Added Usernames, Salts, SHA-256 hash, registration date, and DoB fields
  • md5_hash and sha1_hash have been changed to md5 and sha1 respectively

  • Fixed seed generation bug
